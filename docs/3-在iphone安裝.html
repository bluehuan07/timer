Part 3 — 安裝

在 iPhone 上
當作 App 使用

拿到 GitHub 網址後，透過「加入主畫面」就能把它變成全螢幕、有圖示、像原生 App 的計時器。

一定要用 Safari。iOS 上只有 Safari 能「加入主畫面」，用 Chrome 或其他瀏覽器開這一步會做不出來。
1

用 Safari 開網址

在 iPhone 的 Safari 輸入上一頁拿到的網址：

https://你的帳號.github.io/timer/

確認計時器正常顯示、能按播放。

2

點「分享」

點畫面底部中間的分享按鈕（一個方框、中間往上一支箭頭的圖示）。

3

選「加入主畫面」

在跳出的選單往下滑，找到 加入主畫面（Add to Home Screen），點它。

這時會看到預覽：左邊就是那顆橘環圖示，右邊是名稱「計時器」。按右上角的新增。

完成！

主畫面會出現這顆圖示，點下去就是全螢幕的計時器。

打開後你會發現它跟一般網頁不一樣：

  • 沒有網址列和工具列，整個畫面都是 App（這就是 manifest 裡 standalone 的效果）。
  • 離線也能開。第一次連線載入後，Service Worker 已把檔案快取下來，之後沒網路也照常運作。
  • 有自己的圖示，跟其他 App 排在一起。
兩個 iOS 小限制（先講在前）：
・震動：iPhone 的網頁不支援震動，所以切換階段只有閃光和嗶聲，不會震。
・螢幕休眠：純網頁無法阻止 iPhone 自動鎖屏，倒數放著不動螢幕可能會暗掉。
更新 App：之後你在 GitHub 改了檔案，已安裝的 App 不會馬上更新（因為吃離線快取）。要讓它更新，記得把 sw.js 的版本號 timer-v1 往上加，使用者重開兩次就會拿到新版。
